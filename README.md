# Seafood restaurant

Webpage for a seafood restaurant built with React + TypeScript
## Tech Stack
- React
- Vite
- TypeScript
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/1haut/restaurant.git
cd resto-app
```

Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

The app will run at `http://localhost:5173`.

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
